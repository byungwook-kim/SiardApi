SiardApi - Users' Manual - Introduction

Introduction

SiardApi was developed by Enter AG for the Swiss Federal Archives as part of the SIARD (Software-Independent Archival of Relational Databases) project.

It serves the purpose of providing an API for reading and writing SIARD files conforming to the SIARD Format 2.1.

This document is the users' manual of SiardApi.
